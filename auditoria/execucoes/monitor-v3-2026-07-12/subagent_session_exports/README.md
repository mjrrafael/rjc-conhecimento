# Exportações nativas de sessão — indisponíveis

A plataforma não forneceu export imutável do task graph, prompts integrais, tool calls, paths lidos e outputs de spawn/finalização. As duas ondas foram executadas com canonical task paths distintos, mas a prova negativa exigida não é certificável. A Onda 2 humana registrou leitura acidental de snippets internos ao próprio SHA candidato; por isso a independência falhou materialmente.
